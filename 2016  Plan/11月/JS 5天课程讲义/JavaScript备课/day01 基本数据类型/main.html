数据类型
数据类型 说明 举例
number 包括整数、小数 10,2.0
string 字符串类型,可以用双引号和单引号引住内容（建议使用单引号） "jack"
boolean 只有真、假 false,true
数组 数组可以装任意类型 ['java',10,3.0,false,true]
对象 对象类型，一个key对应一个Value var dog = { name : 'jack',age : 10};

注意点：
JavaScript 拥有动态类型。这意味着相同的变量可用作不同的类型
var x // x 为 undefined
var x = 6; // x 为数字
var x = "Bill"; // x 为字符串
