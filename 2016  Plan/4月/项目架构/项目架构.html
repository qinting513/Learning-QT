项目的制作过程

按照项目的制作过程逐步完成任务. 可以有效集中精力.

  1. 创建工程
  2. 使用Cocoapods引入第三方框架,工程被改为workspace模式
  3. 创建工程目录
  4. 创建pch文件,进行相关配置
  5. 创建constants.h文件,存放常用宏定义
  6. 创建Category文件
  7. Model层制作
  8. 网络请求制作
  9. ViewModel制作
  10. View制作
  11. ViewController中把数据和视图结合展示

项目的目录

一个好的工程目录结构, 可以让工程更加便于维护, 创建下列文件夹, 在不同的文件夹中存储对应的文件

  1. ViewController 视图控制器,即继承于UIViewController的
  2. View 视图,即继承于UIView的
  3. Model 数据模型,放解析类等
  4. Define 放.h头文件, 头文件用于存放宏定义
  5. Resource 图片,音频,视频等等资源,storyboard
  6. ViewModel 视图模型文件
  7. NetManager 网络请求管理类
  8. Category 类别,为其他类增加功能的类
  9. Vendor 第三方类库,通常是Cocoapods无法引入的那些,如第三方分享登录,百度地图,即时通讯等
  10. Supporting Files 放info.plist, pch文件

MJRefresh封装

#import <UIKit/UIKit.h>
#import <MJRefresh.h>

@interface UIScrollView (Refresh)
/** 添加头部刷新 */
- (void)addHeaderRefresh:(MJRefreshComponentRefreshingBlock)block;
/** 添加脚部自动刷新 */
- (void)addAutoFooterRefresh:(MJRefreshComponentRefreshingBlock)block;
/** 添加脚步返回刷新 */
- (void)addBackFooterRefresh:(MJRefreshComponentRefreshingBlock)block;
/** 结束头部刷新 */
- (void)endHeaderRefresh;
/** 结束脚部刷新 */
- (void)endFooterRefresh;
/** 开始头部刷新 */
- (void)beginHeaderRefresh;
/** 开始脚部刷新 */
- (void)beginFooterRefresh;
@end


#import "UIScrollView+Refresh.h"

@implementation UIScrollView (Refresh)
/** 添加头部刷新 */
- (void)addHeaderRefresh:(MJRefreshComponentRefreshingBlock)block{
    self.mj_header = [MJRefreshNormalHeader headerWithRefreshingBlock:block];
}
/** 添加脚部自动刷新 */
- (void)addAutoFooterRefresh:(MJRefreshComponentRefreshingBlock)block{
    self.mj_footer = [MJRefreshAutoNormalFooter footerWithRefreshingBlock:block];
}
/** 添加脚步返回刷新 */
- (void)addBackFooterRefresh:(MJRefreshComponentRefreshingBlock)block{
    self.mj_footer = [MJRefreshBackNormalFooter footerWithRefreshingBlock:block];
}
/** 结束头部刷新 */
- (void)endHeaderRefresh{
    [self.mj_header endRefreshing];
}
/** 结束脚部刷新 */
- (void)endFooterRefresh{
    [self.mj_footer endRefreshing];
}
/** 开始头部刷新 */
- (void)beginHeaderRefresh{
    [self.mj_header beginRefreshing];
}
/** 开始脚部刷新 */
- (void)beginFooterRefresh{
    [self.mj_footer beginRefreshing];
}
@end
